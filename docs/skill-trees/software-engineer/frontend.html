前端工程師

備註

資深（Senior）

資深前端工程師 (Senior Front-end Engineer)​

核心技術能力 (Technical Skills)​

  • 前端基礎與進階技術：
    • 精通 HTML5, CSS3, JavaScript (ES6+)。
    • 熟悉 TypeScript。
    • 熟悉 前端模組化開發、響應式設計 (RWD)、以及跨裝置/瀏覽器相容性調整。
  • 主流框架與生態系：
    • 具備 4 年以上前端開發經驗，並有獨立開發與主導專案的能力。
    • 熟悉至少一種主流前端框架，尤其是 React.js 生態系（包含 Hooks、Context、Router 等現代開發方式），並具備大型專案開發經驗。
  • 建構與工具鏈 (Tooling)：
    • 熟悉 Webpack, Vite, Rollup 等建構工具。
    • 具備 CI/CD 使用經驗。
  • 版本控制與協作 (Version Control & Collaboration)：
    • 熟悉 Git 版本控制（以 GitHub Flow 為佳），具備多人協作經驗。
    • 具備良好的 Code Review 習慣。

程式碼品質與架構 (Quality & Architecture)​

  • 程式碼素養：
    • 重視程式碼品質與可維護性，具備撰寫乾淨、可讀性高程式碼的習慣。
    • 具備規劃與重構前端架構的能力，以提升可維護性與擴展性。
  • 使用者體驗 (UX)：
    • 具備優化效能、提升使用者互動體驗的能力。
    • 能與設計人員一同優化產品 UI/UX 體驗。

軟實力與溝通 (Soft Skills & Communication)​

  • 溝通與合作：
    • 擁有良好的溝通能力與團隊合作精神。
    • 能主動釐清需求、理解問題，並將其轉化為具體的技術解決方案。
    • 樂於分享經驗與洞見，協助團隊成員提升技術能力。

加分條件 (Bonus Qualifications)​

  • UI/Design 框架：
    • 熟悉任一 UI Framework (如 Tailwind CSS, MUI, Ant Design 等)。
  • 測試經驗：
    • 具備前端自動化測試經驗 (如 Jest, Testing Library, Cypress 等)。
  • 後端與 API 協作：
    • 有前後端串接經驗，能與後端 API 有效整合，甚至能參與制定 API 相關經驗。
  • 網站優化：
    • 有 SEO, GA (Google Analytics), GTM (Google Tag Manager) 或 Lighthouse 性能優化經驗。
  • 雲端與部署：
    • 熟悉雲端環境特性。
    • 有前端部署流程 (如 Vercel, Netlify) 經驗。
  • 系統規劃：
    • 具備在既有產品中導入更高效的 UI 實作方式與元件設計模式的經驗。
