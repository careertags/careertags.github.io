# 名言金句

> 「你的職涯道路，一定要是直線向上的嗎？拋開階梯，踏入蜿蜒（Squiggle）職涯。」

# 前言

這次要跟大家分享 Sarah Ellis 與 Helen Tupper 在 TED 上的演講：「最好的職涯道路並不總是直線向上」（The best career path isn't always a straight line）。兩位講者透過自身經歷與觀察，挑戰傳統「職涯階梯」（Career Ladder）的線性發展觀念，提出「蜿蜒職涯」（Squiggly Career）的概念。在現今快速變化的世界中，職涯不再是單一、固定的路徑，而是充滿不確定性與可能性的蜿蜒旅程。傳統階梯限制了學習與機會，並已成為過時的概念，尤其在只有 6% 的人仍從事朝九晚五工作的今日，它更無法適應未來的需求。兩位講者鼓勵個人重新定義成功、主動規劃學習，並探索多樣化的職涯可能性，強調在蜿蜒職涯中，每個人都能透過發揮才能找到屬於自己的成功之路。

# 重點摘要

1.  階梯職涯的限制與過時：

    - 傳統職涯階梯的概念源於 1900 年代，強調一致性與控制，但這與現今不斷變化的工作世界已不相符。
    - 階梯限制了個人學習與機會，且若組織不放棄階梯，將可能流失人才。
    - 現代人預計會經歷五種不同類型的工作，且 50% 的現有技能在 2025 年將不再適用。

2.  蜿蜒職涯的核心精神：

    - 蜿蜒職涯充滿不確定性，但也充滿可能性，變化隨時都在發生 。
    - 成功的定義是個人化的，每個人都有獨特的職涯路徑。

3.  重新定義學習關係：

    - 擺脫階梯要從重新定義我們與「學習」的關係開始，每個人都有機會策劃自己的學習藍圖。
    - 學習不再受限於組織層級或特定部門，也不僅發生在參加課程時。
    - 在蜿蜒職涯中，每個人都是學習者，同時也是老師，可以透過相互學習成長。

4.  改變對晉升的看法：

    - 職涯階梯只往一個方向走，將晉升純粹等同於升職會讓人錯失許多機會。
    - 應停止只問「下一個工作是什麼」，而是問「我對探索哪些職涯可能性感到好奇」。

5.  擁抱與利用個人才能：
    - 在階梯職涯中，個人身份常被職稱所定義，而非擁有的才能。
    - 每個人都擁有才能，且能以多種方式運用這些才能，不必將職涯受限於單一模式。

# 行動方案

1.  策劃個人學習藍圖：主動選擇適合自己的學習資源，例如觀看 TED 演講、閱讀書籍或部落格、收聽 Podcast 等，將發展掌握在自己手中。
2.  探索職涯可能性：給自己許可去探索感興趣的、甚至是不覺得準備好的或曾被放棄的職涯選項。
3.  尋求組織支持與環境：鼓勵所屬組織或部門建立類似 Cook 公司的「夢想學院」等計畫，支持同事探索組織內外的各種職涯。
4.  展現真實自我與獨特性：在工作中「做你喜歡的事，以真實面貌示人」，將獨特性帶入組織，不必迎合特定模式。

# 問題反思

1. 你如何界定自己職涯中的「成功」，它是否仍然是傳統的升職和加薪？
2. 在你目前的工作環境中，職涯階梯的概念如何限制了你學習或發展新技能的機會？
3. 你可以採取什麼具體行動，來重新策劃自己的學習路徑，而不依賴公司提供的培訓？
4. 你曾經放棄的、但現在仍感興趣的職涯可能性是什麼？你會如何開始探索它？
5. 作為一名管理者或導師，你如何能成為「蜿蜒職涯」的倡導者，幫助他人探索他們的可能性？

# 總結

Sarah Ellis 和 Helen Tupper 提出「蜿蜒職涯」的概念，是對單向、線性「職涯階梯」的強烈挑戰。面對當今變動快速的工作環境，職涯發展需要從外部標準轉向個人好奇心與才能。這場演講的關鍵在於呼籲一個激進的「徹底重做」，鼓勵個人拿回職涯的控制權，透過持續學習、探索多樣化的可能性來定義自己的成功。這不僅是對個人的呼籲，也是對組織的警示：擁抱蜿蜒，才能留住那些不斷適應、好奇、自信和持續學習的員工。是時候「走出階梯，邁入蜿蜒」了。
